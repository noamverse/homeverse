# OG Image Instructions

Create a 1200×630px image at `/public/og-default.png` with:

- Dark warm charcoal background matching the site (`#0f0e0d` or `#141210`)
- HOME logo centered or left-positioned with multi-color gradient glow (blue → purple → orange)
- Headline text: "Life is a family, not a marketplace."
- Typography: Playfair Display for the headline, Inter for supporting text
- Subtle ambient atmosphere (warm/cool radial blooms at low opacity)
- Optional: wordmark "HOME" in large Playfair at top or bottom

Until this file exists, OG previews will fail gracefully — the site will still work,
but links shared on social media will not show a rich card preview.

Recommended tools: Figma, Canva, or any image editor that supports custom canvas sizes.
Export as PNG at 1200×630px.
